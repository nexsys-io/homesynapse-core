# API Specifications

This directory contains the spec-first API definitions for HomeSynapse Core.

## Files

- `openapi/` — OpenAPI 3.1 specification for the REST API (Doc 09, produced in Phase 2 Wave 4)
- `asyncapi/` — AsyncAPI 3.0 specification for the WebSocket API (Doc 10, produced in Phase 2 Wave 4)

## Validation

Specs are validated as part of the CI pipeline. See the corresponding design
documents in `homesynapse-core-docs` for the full API design rationale.
